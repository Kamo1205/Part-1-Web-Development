# Baker's Delights - Bakery Website

## Overview
This is a single-page static website for **Baker's Delights**, a family-owned bakery located in Polokwane, South Africa. The site showcases the bakery's offerings including freshly baked pastries, artisan breads, cakes, and coffee. It features a modern, responsive design with smooth navigation.

## Features
- **Hero Section**: Eye-catching introduction with call-to-action.
- **About Us**: Story of the bakery and its passion for quality baking.
- **Menu**: Interactive menu grid displaying 6 popular items:
  | Item              | Price     | Description                          |
  |-------------------|-----------|--------------------------------------|
  | Butter Croissants | R35       | Flaky, golden croissants             |
  | Chocolate Cake    | R180      | Rich moist cake with ganache         |
  | Sourdough Bread   | R30       | Handcrafted with natural levain      |
  | Vanilla Cupcakes  | R20 each  | Light cupcakes with buttercream      |
  | Fresh Coffee      | R25       | Premium roasted coffee               |
  | Artisan Bread     | R40       | Traditional white bread              |
- **Contact Section**: Location, phone (+27 87 557 8765), email (hello@bakersdelights.co.za), and business hours.
- **Responsive Navigation**: Hamburger menu for mobile devices.
- **Google Fonts**: Playfair Display and Poppins for elegant typography.

## Location
- **Address**: 123 Bakery Street, Polokwane, 0699
- **Business Hours**: Mon-Sat: 7am - 5pm | Sun: 8am - 2pm

## Files
- `index.html`: Main website (primary version)
- `README.md`: This file

## How to Run
1. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
2. No server or build tools required - fully static.
3. Connect to internet for Google Fonts to load.

## Technologies Used
- **HTML5**: Semantic structure
- **JavaScript**: None (hamburger menu may need JS for functionality)
- **Google Fonts**: Typography
